## WRITING BLOCK (Lessons 4-7)

### L04: Short Notes Win ✅ CREATED

### L05: Bullet Power
**Flow:**
1. Intro: When to use bullets (lists, ideas, steps)
2. Example: Paragraph → Bullet list transformation
3. Practice: Convert shopping list paragraph to bullets
4. Practice: Recipe instructions → numbered bullets
5. Rule: Use bullets for separate items, numbers for order
6. Summary: Bullets organize information

**Key Concepts:**
- Bullets for unordered items
- Numbers for sequential steps
- One idea per bullet
- Keep bullets parallel (same format)

**Practice P05A:** Convert 3 paragraphs to bullet lists
**Practice P05B:** Choose bullet vs number format for 5 scenarios

## RESEARCH-BASED DESIGN PRINCIPLES

**Cognitive Load Theory**
- One concept per step
- Progressive complexity
- Worked examples before practice

**Spaced Repetition**
- Reviews after 1 day, 3 days, 1 week
- Mixed practice in reviews
- Retrieval practice built-in

**Dual Coding**
- Visual + verbal processing
- Symbols + text
- Examples + explanations

**Generation Effect**
- Students create notes, not copy
- Build own examples
- Personalize systems

**Metacognition**
- Explicit strategy instruction
- Self-evaluation opportunities
- Quality assessment
